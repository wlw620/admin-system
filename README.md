
``` bash
# 安装 node

# install dependencies
npm install

# serve with hot reload at localhost:3000
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# 构建后的静态文件都打包在dist下

# src 源文件

# src => i18n 语言包文件
# src => config 一些静态配置文件
# src => store 单项数据流 全局状态管理
# src => static 静态资源目录
# src => pages 个页面
# src => components => 组件


```
